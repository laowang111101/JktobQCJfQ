# 电影推荐系统 python721

## 项目概述

本项目是一个基于Python的协同过滤算法的电影推荐系统，旨在为用户提供个性化的电影推荐服务。

## 技术栈

- **后端技术：**
  - Python
  - Django框架

- **前端技术：**
  - VUE

- **开发工具：**
  - PyCharm

## 功能模块

### 前台管理

- 电影信息
- 电影资讯
- 个人中心
- 电影搜索
- 电影收藏
- 评分

### 后台管理

- 个人中心
- 电影分类管理
- 用户管理
- 电影评分管理
- 系统管理

## 系统角色

- 普通用户：可以浏览电影信息，进行电影搜索、收藏、评分等操作。
- 管理员：负责管理电影分类、用户信息、电影评分以及系统设置等。

## 运行环境

- 后端：Python 3.x，Django 2.x
- 前端：VUE.js
- 数据库：MySQL

## 部署步骤

1. 安装Python环境。
2. 安装Django框架。
3. 配置数据库MySQL。
4. 部署前端VUE.js代码。
5. 运行后端服务。
